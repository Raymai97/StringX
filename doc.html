FunctionReturn value

strxCount Number of key contained in str
strxIndexOf Index of FIRST key in str
strxLastIndexOf Index of LAST key in str
strxSplit Number of splitted strings
strxCat String concatenated from multiple strings
strxSubstr String extracted from index A for B chars
strxUpper and strxLower String in UPPER or lower case
