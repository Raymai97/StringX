StringX
introduction documentation get on github
A free lightweight string processing library for C

  • What does StringX provides:

    • strxIs - Compare string
    • strxHas - Check if string contain keyword
    • strxCount - Count occurences of keyword
    • strxIndexOf - Get index of the FIRST keyword
    • strxLastIndexOf - Get index of the LAST keyword
    • strxSplit - Split string into string array
    • strxCat - Concatenate multiple strings to one
    • strxSubstr - Extract the string from index A to B
    • strxUpper and strxLower - Get string in UPPERCASE / lowercase

  • Why StringX?

    • Have you ever tried to code like this? 
    				if (strcmp(input, "exit")) // oops! missing '== 0'...
				{ ...
    With StringX, it will be 
    				if (strxIs(input, "exit", 0)) // elegant!
				{ ...
    where the 3rd argument can be replaced with STRX_FLAGS. 
    				STRX_FLAGS f = STRX_CASE_INSENSITIVE; // most users aren't case-sensitive
				if (strxIs(input, "exit", f)) {...}
				else if (strxIs(input, "help", f)) {...}
    With STRX_FLAGS, many things work wonder. 
    				// If contain the ENTIRE keyword...
				if (strxHas(input, F_BOMB_WORD, f)) {...}

				// If contain ONE of the keyword...
				if (strxHas(input, INVALID_CHARS, f | STRX_KEY_AS_SINGLE_CHAR)) {...}
    StringX also supports both ASCII and UTF-16 char type.
    To enable wchar_t support, just define _STRINGX_USE_WCHAR in compiler settings.

  • StringX is eXtension of string.h, not replacement

    • StringX doesn't completely reinvent the wheel. For example, why would one re-do the strlen()?
    The stringx.h includes string.h, so it is like string.h extension.
